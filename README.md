### :cloud: Installation

**Only one step :**
```shell
composer require meilleursbiens/immodvisor-api-wrapper
```

```php
$api = new Meilleursbiens\ImmodvisorApiWrapper\ImmodvisorAPI("API_TOKEN", "CHECKSUM_IN", "CHECKSUM_OUT");
$api->getClient()->activities();
```
